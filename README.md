> یک ربات تلگرام پیشرفته با سیستم بازی‌های متنوع، سطح‌بندی، و دستاوردها

## ✨ ویژگی‌ها

### 🎯 بازی‌های متنوع
- 📦 **صندوق‌های گنج** - احتمال‌های متفاوت برای پیروزی و شکست
- 🎲 **رول تاس** - شانس ۵۰/۵۰ برای برنده شدن
- ✨ **شماره خوشبخت** - انتخاب ۱ از ۱۰ شماره با پاداش ۵ برابر
- 🪙 **پرتاب سکه** - شیر یا خط؟

### 👤 سیستم کاربران
- 💰 موجودی و نقاط
- 🎖️ سطح‌بندی (۵ سطح از شروع‌کننده تا افسانه‌ای)
- 📊 آمار تفصیلی (برد، باخت، درصد پیروزی)
- 🏅 دستاوردهای قابل باز کردن

### 🏆 رقابتی
- 🏆 جدول امتیازات (لیدربورد)
- 📈 آمار زندگی‌کاربر
- 🎁 جایزه روزانه (۵۰ سکه هر ۲۴ ساعت)

### 🛡️ سیستم‌های پیشرفته
- 🔒 مدیریت امن داده‌ها
- 📝 ثبت تراکنش‌ها
- 🔄 سطح‌بندی خودکار
- ⚡ کارایی بالا با Threading

## 🚀 شروع به کار

### پیش‌نیاز‌ها
- Python 3.9 یا بالاتر
- pip (مدیر بسته‌های Python)

### نصب و اجرا

1. **کلون کردن مخزن:**
```bash
git clone https://github.com/Amirdev-kb/telegram-bot.git
cd telegram-bot
```

2. **نصب وابستگی‌ها:**
```bash
pip install -r requirements.txt
```

3. **تنظیم توکن:**

**گزینه ۱: فایل `.env`**
```bash
echo "BOT_TOKEN=YOUR_TOKEN_HERE" > .env
```

**گزینه ۲: فایل `config.py`**
```python
BOT_TOKEN = "YOUR_TOKEN_HERE"
```

4. **اجرای ربات:**
```bash
python bot.py
```

## 💾 پایگاه داده

ربات از SQLite استفاده می‌کند:

### جداول
- **users** - اطلاعات کاربران
- **plays** - تاریخچه بازی‌ها
- **achievements** - دستاوردهای کاربران
- **transactions** - ثبت تراکنش‌ها

## 🔧 تنظیمات پیشرفته

به فایل `config.py` مراجعه کنید برای تنظیمات:
- `GAME_CONFIG` - تنظیمات بازی و پاداش
- `LEVEL_CONFIG` - سطح‌های بازی
- `BOT_CONFIG` - تنظیمات ربات
- `DB_CONFIG` - تنظیمات پایگاه داده
