فایل‌های اضافه شده:
- bot.py — ورودی و هندلرها
- storage.py — SQLite helper (بهبودها: زمان‌بندی بازی و توابع مدیریتی)
- game.py — منطق بازی
- ui.py — کیبوردها و منوها
- config_example.py — نمونه تنظیمات (افزوده شدن ADMINS)
- requirements.txt
- Dockerfile — برای اجرا در کانتینر
- .gitignore

نحوه اجرا (محلی):
1. پایتون 3.9+ نصب کن.
2. یک فایل `config.py` بساز و `BOT_TOKEN` رو قرار بده یا متغیر محیطی ست کن. اگر از دستورات مدیریتی استفاده می‌کنی، ADMINS را هم در config.py اضافه کن.
3. نصب وابستگی‌ها:
   pip install -r requirements.txt
4. اجرای بات:
   python bot.py

نکته‌ها و بهبودها:
- محدودیت زمانی بین بازی‌ها (cooldown) اضافه شد تا از اجرای سریع و پشت‌سرهم جلوگیری شود.
- فرمان مدیریتی /setbalance برای تنظیم موجودی یک کاربر اضافه شد (تنها برای ADMINS).
- Dockerfile برای استقرار ساده اضافه شد.

حریم خصوصی و ایمنی:
- دیتابیس محلی SQLite است. در صورت انتشار در سرور، مسیر DB و دسترسی‌ها را ایمن کنید.
