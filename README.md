# Telegram Bot — MiniApp + Game (Coin Collector)

این پروژه نسخه‌ای توسعه یافته از ربات تلگرام است که یک مینی‌اپ بازی ساده (Coin Collector) اضافه می‌کند.

فایل‌های اضافه شده:
- bot.py — ورودی و هندلرها
- storage.py — SQLite helper
- game.py — منطق بازی
- ui.py — کیبوردها و منوها
- config_example.py — نمونه تنظیمات
- requirements.txt

نحوه اجرا (محلی):
1. پایتون 3.9+ نصب کن.
2. یک فایل `config.py` بساز و `BOT_TOKEN` رو قرار بده یا متغیر محیطی ست کن.
3. نصب وابستگی‌ها:
   pip install -r requirements.txt
4. اجرای بات:
   python bot.py

توضیحات بازی:
- هر کاربر هنگام اولین اجرای ربات 100 سکه دریافت می‌کند.
- با فشردن مینی‌اپ، کاربر می‌تواند یکی از 3 صندوق را انتخاب کند؛ نتیجه بر اساس احتمالات تعیین می‌شود و ممکن است جایزه یا جریمه باشد.
- موجودی و تاریخچه‌ی بازی‌ها در SQLite ذخیره می‌شود.
- دستورها: /menu، /miniapp، /balance، /leaderboard، /daily

حریم خصوصی و ایمنی:
- دیتابیس محلی SQLite است. در صورت انتشار در سرور، مسیر DB و دسترسی‌ها را ایمن کنید.
